# simple_cpp_remote

Same as simple_cpp_local, but with a different
version number, and a repository
served from github.io.




* group id: org.team100.simple_cpp
* artifact id: simple_cpp
* version: 0.0.2


```
zip -r simple_cpp-0.0.2-linuxx86-64.zip simple_cpp-0.0.2-linuxx86-64
```

Then I copied the zip and pom to the repository directory.

```
$ tree repository
repository

    └── team100
        └── simple_cpp
            └── simple_cpp
                └── 0.0.2
```
